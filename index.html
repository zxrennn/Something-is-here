// Instructions

How to play

Movement
Move through the maze
Navigate dark corridors to find orbs and avoid the monster.
WASDor arrows
Survivor
Collect all orbs
Purple orbs are scattered each level. Grab them all to advance. Hiding in dark zones makes you invisible to the monster.
Co-op Escape
Find 3 keys, reach the exit
All players work together. Collect enough keys to unlock the exit, then reach it before the timer ends.
Versus
Monsters vs Survivors
2 monsters hunt 3 survivors. Survivors collect orbs for points. Monsters catch survivors. Whoever dominates wins.
The monster AI
Three-state hunter
Patrol — sweeps all rooms.
Search — hunts last known position.
Chase — full sprint. ! = it sees you.
Shop
Earn coins, buy cosmetics
Coins drop per orb collected and per level. Buy skins, trails, and chat titles. Press E to interact with the exit in escape mode.

// Dev Log

Patch Notes

v2.1.0Mar 2026
Patches & Fixes
  • Fixed wall collision — players can no longer walk through walls
  • Level 1–3 now has fewer rooms and walls — map gets denser as you progress
  • Level locking — must complete a level to unlock the next one
  • Monster proximity chase trigger reduced — it only reacts when you're very close, not just touching its sight circle
  • Mobile d-pad auto-detected via pointer type and screen width
  • Orb count scales per level (starts lower, ramps up)
Mar 2026
Multiplayer Update
  • Full real-time multiplayer via WebSockets (up to 5 players)
  • Create rooms with unique 6-char codes — share to invite friends
  • Join rooms with code — custom player names with NSFW filter
  • Co-op Escape gamemode — collect 3 keys, unlock exit, escape together
  • Versus gamemode — 2 monsters vs 3 survivors, catch or outlast
  • In-game chat during lobby and gameplay
  • Cosmetic shop — skins, trails, chat titles bought with earned coins
  • Coins earned per orb collected and per level completed
  • Multiplayer scoreboard on game over
  • AI human in monster mode now flees when you get close
  • Devlog added to website
v1.0.0Mar 2026
Initial Release
  • First playable build — survive, hide, collect orbs
  • 9-room maze with carved corridors and dark zones
  • Three-state monster AI: patrol, search, chase with proximity trigger
  • Sanity meter, procedural audio, leaderboard
  • Pause menu, Esc key, mobile d-pad
  • Monster spawns in farthest room, never inside walls
  • Play as Monster mode with fog of war
  • Level select (1–9) with scaling difficulty
* more to come
